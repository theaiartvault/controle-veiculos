# Controle de Veículos — Portaria

App para registrar manualmente a entrada e saída de veículos quando a TAG ou a leitura de placa falhar.

## Como publicar no GitHub Pages

1. Crie um repositório novo (ex: `controle-veiculos`) na sua conta `theaiartvault`.
2. Suba todos os arquivos desta pasta para a raiz do repositório.
3. Em Settings → Pages, selecione a branch `main` e a pasta `/ (root)`.
4. Acesse `https://theaiartvault.github.io/controle-veiculos/`.

## Como instalar no celular/tablet da portaria

1. Abra o link no Chrome (Android) ou Safari (iPhone/iPad).
2. Vai aparecer um banner "Instalar" — toque nele. (No Safari: toque em Compartilhar → Adicionar à Tela de Início.)
3. O app abre como um aplicativo normal, em tela cheia, com ícone próprio.
4. Depois de aberto uma vez com internet, ele continua funcionando 100% sem internet — todos os dados ficam salvos no próprio aparelho.

## Como usar

- **Novo registro**: escolha Morador ou Visitante, toque na torre (1 a 9), digite apto, nome e placa, e toque em "Registrar entrada".
- **No pátio**: lista de veículos que ainda não saíram. Toque em "Marcar saída" quando o veículo for embora.
- **Histórico**: todos os registros, com busca por placa/torre/apto/nome. Pode exportar um PDF do turno ou limpar os dados.

## Importante

- Os dados ficam salvos *apenas* no aparelho onde o app foi usado (não há servidor/nuvem). Se for usar em mais de um aparelho, os registros não se misturam automaticamente — exporte o PDF no fim do turno para manter um histórico único.
- Se quiser zerar a lista pra um novo turno, use "Limpar tudo" depois de exportar o PDF.
